
**Built:**
•	PDF ingestion pipeline
•	NLP text preprocessing
•	TF-IDF vectorization
•	Cosine similarity scoring
•	Skill gap analytics

Input: Resume (pdf) &	A job description (text)

Job Data → Embeddings → KMeans → Cluster Groups → Salary Aggregation → Salary Intelligence Agent → Resume Salary Prediction

**Analyser UI Dashboard:**
A modern, single-page web application that analyzes resumes against job descriptions and provides ATS-friendly scores, skill gap analysis, and improvement suggestions **using Visual Studio Code**. Perfect for job seekers who want to optimize their resumes and improve hiring chances.

How It Works
1. Login with demo credentials (admin / password)
2. Upload your resume PDF and paste the job description
3. The system analyzes:
ATS score (how well your resume matches the job)
Matched vs missing skills
AI-generated improvement suggestions
4. Visualize results in bar charts and pie charts
5. Download PDF report with full analysis
